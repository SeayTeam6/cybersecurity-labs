## Objective
Demonstrate foundational Azure skills by creating and managing cloud resources, uploading and modifying data in Azure Storage, and properly cleaning up resources to prevent unnecessary costs.

## Tools Used
- Microsoft Azure Portal
- Azure Resource Groups
- Azure Storage Account
- Local text file (.txt)

## Lab Summary
Created an Azure Resource Group and Storage Account, uploaded and edited a text file directly within the Azure Portal, then downloaded and verified the changes locally. Practiced proper cloud cost management by deleting all resources and validating cleanup through Azure Cost Management and Resource Group verification.
